Detailed Description

This Logger class deals with logging one severity of message.

It can also be used to specify the category of the message, using operator[].

Logger instances instantiate LogMessage objects, which when destroyed, flushes the message to the log.

LogMessage objects lazily evaluate their arguments, so that no computation overhead is incurred if the message is not used (because of filtering)
