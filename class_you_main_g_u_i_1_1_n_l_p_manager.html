Detailed Description

The component that handles passing of data between the NLP parser and the rest of the GUI.

It has two primary operations: on setup it passes a list containing the IDs of tasks that were previously open, and receives a Result object. Every time a command is sent, the current Result as well as input string is sent to the parser. The Result is a vector of tasks, and this replaces the existing Result. It inherits from the BaseManager class.
